For the spotify code, in the ./Spotify/ folder, you need a ClientID.txt, ClientSecret.txt, and Redirect.txt file which is used for the spotify api
\
And for the actual server you need a file called logins.json file in the form of {username:string,password:string}[]\
\
Modular online control system. A server which will runs can be connected to by any kind of device. A device which has gives the server access to do something can be controlled in real-time through websocket or through the web server running on the main server. The web server can pass in values and call functions through the server to the device.
